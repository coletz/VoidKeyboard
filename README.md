# VoidKeyboard

This is a simple keyboard with no auto-completion nor auto-correction for the Minimal Phone MP01.

Features:
- CTRL actions: hold SYM key and press Z/C/V to undo/copy/paste (and every other shortcut)
- SYM layer (to be improved): press the SYM key to open the SYM layer (open an issue to request new keys)
- Shift/Caps lock: hold shift key and press other keys to write uppercase chars; double press shift to enable caps lock
- ALT key: hold ALT key and press other keys to write alternative characters
- Support for multiple languages alternative characters (open an issue to request a new language)

Download of the prebuild APK is available in the [release tab](https://github.com/coletz/VoidKeyboard/releases/latest)
